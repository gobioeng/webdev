HALog is currently in beta testing. We're actively seeking feedback from radiotherapy engineers and medical physicists to improve the software before its official release. While functional, you may encounter bugs or incomplete features. We appreciate your understanding and welcome your feedback.

What is HALog?

HALog is a specialized desktop application designed to help radiotherapy engineers and medical physicists monitor and analyze water cooling systems in linear accelerators (LINACs), ensuring optimal performance and detecting potential issues before they affect patient treatment.

Real-time monitoring of key water system parameters
Advanced analytics with trend detection and anomaly identification
Efficient processing of large log files with progress tracking
Support for multiple log formats from different LINAC systems
Professional UI with dynamic trend plots and statistical distribution charts

Key Features

Advanced Analytics

Comprehensive statistical analysis with confidence intervals, trend detection, and anomaly identification for optimal cooling system performance.

Real-time Monitoring

Track key water parameters including pump pressure, magnetron flow, target and circulator flow, and city water flow with real-time updates.

Efficient Data Processing

Chunked file processing handles large log files (67K+ lines) with progress tracking and non-blocking UI during data analysis operations.

Anomaly Detection

Advanced anomaly detection algorithms (DBSCAN clustering and Isolation Forest) identify potential issues before they affect patient treatment.

Professional UI

Intuitive user interface with dynamic trend plots, statistical distribution charts, and anomaly highlighting for easy analysis and interpretation.

Multiple Log Formats

Automatically detects and parses various parameter formats, ensuring compatibility with different LINAC systems and their logging formats.

System Requirements

Hardware Requirements

  • Windows 10/11 (64-bit recommended)
  • 2GB RAM minimum (4GB recommended)
  • 500MB free disk space
  • Screen resolution: 1280x720 or higher
  • Internet connection (for updates)

Software Dependencies

  • Python 3.8 or higher (bundled with installer)
  • PyQt5 for user interface
  • Pandas for data processing
  • Matplotlib for data visualization
  • SciPy and scikit-learn for analytics

Supported Log Formats

  • Standard timestamp: YYYY-MM-DD HH:MM:SS
  • Multiple parameter name formats
  • Statistics with count, min, max, and avg
  • Plain text log files (.txt, .log)
  • Multiple serial number formats

Frequently Asked Questions

What is HALog used for?

HALog is a professional desktop application designed for monitoring and analyzing water system parameters in Linear Accelerator (LINAC) machines used in radiotherapy. It helps medical physicists and biomedical engineers track key metrics like pump pressure, magnetron flow, target flow, and city water flow to ensure optimal system performance and prevent issues that could affect patient treatment.

Is HALog free to use?

Yes, HALog is currently available as a free beta (version 0.0.1) for medical professionals, researchers, and educational institutions. We're actively gathering feedback to improve the software before its official release.

How do I import log files into HALog?

Importing log files is simple. Launch HALog and select "Open Log File" from the File menu (or press Ctrl+O). Navigate to your log file and select it. HALog will automatically process the file, displaying a progress bar during import. Once complete, you'll see a summary of the imported data and can begin analyzing it through the dashboard, trends, and analysis tabs.

Can HALog process large log files?

Yes, HALog is designed to handle large log files efficiently. It uses chunked processing and background threading to maintain UI responsiveness even when processing files with 67,000+ lines. The application also includes progress tracking with real-time updates and estimated completion time.

What analytics capabilities does HALog offer?

HALog provides comprehensive analytics including statistical analysis (mean, standard deviation, quartiles, coefficient of variation), trend detection using linear regression, anomaly detection using machine learning algorithms (DBSCAN clustering and Isolation Forest), and quality assessment of parameters. These analytics help identify potential issues before they affect system performance.
